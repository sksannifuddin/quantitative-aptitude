# Quantitative Aptitude Preparation — Full Overview

Part of a placement-prep journey running from mid-July 2026 toward placement season, alongside Core (Python → Stats → ML → DL → NLP → GenAI), DSA, Verbal/Logical Reasoning, and English.

## Goal

Cover the complete placement-relevant Quantitative Aptitude syllabus, prioritized by what's actually asked in campus placements and aptitude tests (TCS NQT, Infosys, Accenture, Deloitte, Cognizant, Wipro, Capgemini, banking/CAT/GATE-style aptitude), rather than spreading effort evenly across every possible topic.

## Approach

- Topics are tackled in **priority tiers** (below), so the highest-yield material is solid before spending time on rarer topics.
- Runs alongside Core, DSA, and English in a daily evening study block, with Aptitude getting its own dedicated slot.
- Self-paced from August onward — depth over rushing, same as the other tracks.
